# **App Name**: MoneyMate

## Core Features:

- Days Until Reset Calculation: Automatically calculates the number of days until the user's Copilot quota resets.
- Countdown Timer: Display a countdown timer showing the time remaining until the Copilot quota resets.
- Personalized Encouragement Tool: Use AI to generate a customized, encouraging message to be displayed alongside the quota reset information, providing helpful suggestions based on remaining allowances and optimal copilot usage.
- Usage Visualization: Offer a clear visual indicator (e.g., progress bar) of the user's Copilot usage within the current quota period.
- Persistent Data Storage: Store the reset date in local storage to maintain the countdown across sessions.

## Style Guidelines:

- Primary color: HSL 210, 75%, 50% (RGB Hex: #3399CC) for a dependable, calming feel.
- Background color: HSL 210, 20%, 95% (RGB Hex: #F0F4F7) for a clean, light, calming feel.
- Accent color: HSL 180, 60%, 40% (RGB Hex: #26998F) to complement the primary, and create contrast.
- Body and headline font: 'Inter' sans-serif for a modern and neutral feel.
- Minimalist design with a focus on clarity and ease of use. Clean typography and ample white space to improve readability.
- Use simple, outline-style icons to represent key features and information.
- Subtle transitions and animations for visual feedback during data updates or user interactions.
